# react-autocomplete-france

Headless React hook for French address autocomplete powered by [Géoplateforme](https://data.geopf.fr/geocodage).

**Zero UI opinions** — bring your own components and styles (MUI, Ant Design, shadcn/ui, Tailwind, plain HTML...).

## Installation

```bash
npm install react-autocomplete-france
```

## Basic usage

```tsx
import { useAutocomplete } from 'react-autocomplete-france'

function AddressInput() {
  const { query, setQuery, suggestions, isLoading, isOpen, selectSuggestion } = useAutocomplete({
    onSelect: (suggestion) => console.log(suggestion),
  })

  return (
    <div>
      <input value={query} onChange={(e) => setQuery(e.target.value)} placeholder="Rechercher une adresse..." />
      {isOpen && (
        <ul>
          {isLoading
            ? <li>Chargement...</li>
            : suggestions.map((s) => (
                <li key={s.properties.id} onClick={() => selectSuggestion(s)}>
                  {s.properties.label}
                </li>
              ))
          }
        </ul>
      )}
    </div>
  )
}
```

## Hook options

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `onSelect` | `(suggestion: AutocompleteFeature) => void` | — | Called when a suggestion is selected |
| `debounce` | `number` | `300` | Debounce delay in ms |
| `limit` | `number` | `5` | Max number of suggestions |
| `searchParams` | `Partial<AutocompleteQueryParams>` | — | Additional API search params (postcode, type, etc.) |
| `baseURL` | `string` | `https://data.geopf.fr/geocodage` | Custom API base URL |

## Hook return value

| Property | Type | Description |
|----------|------|-------------|
| `query` | `string` | Current search query |
| `setQuery` | `(value: string) => void` | Update the search query |
| `suggestions` | `AutocompleteFeature[]` | List of suggestions |
| `isLoading` | `boolean` | Whether the API request is in progress |
| `isOpen` | `boolean` | Whether suggestions should be displayed |
| `selectSuggestion` | `(suggestion) => void` | Select a suggestion |
| `clear` | `() => void` | Clear query and suggestions |

## Advanced: direct API usage

```ts
import { AutocompleteAPI } from 'react-autocomplete-france'

const api = new AutocompleteAPI()
const results = await api.autocompleteAddress('10 rue de la paix', 5)
```
